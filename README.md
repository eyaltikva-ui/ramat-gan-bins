# ספירת פחי מיחזור - רמת גן 🗑️

אפליקציה לספירת פחי מיחזור כחולים ברחובות רמת גן עם GPS אוטומטי ודשבורד ניהול.

## קבצים באפליקציה

- `index.html` - דף בית (מפנה אוטומטית ל-app.html)
- `app.html` - אפליקציית הספירה (לעובדים בשטח)
- `dashboard.html` - דשבורד ניהול (למנהל הפרויקט)

## הוראות העלאה ל-GitHub Pages

### שלב 1: יצירת חשבון GitHub (אם אין לך)
1. היכנס ל-https://github.com
2. לחץ על "Sign up"
3. צור חשבון חינמי

### שלב 2: יצירת Repository חדש
1. לחץ על הכפתור הירוק "New" או "+" בפינה הימנית העליונה
2. בחר "New repository"
3. תן שם ל-Repository: `ramat-gan-bins`
4. סמן את התיבה "Public"
5. סמן את התיבה "Add a README file"
6. לחץ "Create repository"

### שלב 3: העלאת הקבצים
1. בעמוד ה-Repository, לחץ על "Add file" ובחר "Upload files"
2. גרור את 3 הקבצים:
   - `index.html`
   - `app.html`
   - `dashboard.html`
3. כתוב הודעה למטה: "Initial commit"
4. לחץ "Commit changes"

### שלב 4: הפעלת GitHub Pages
1. בעמוד ה-Repository, לחץ על "Settings" (למעלה)
2. בתפריט הצדדי השמאלי, לחץ על "Pages"
3. תחת "Source", בחר "Deploy from a branch"
4. תחת "Branch", בחר "main" ולחץ "Save"
5. המתן כ-2-3 דקות

### שלב 5: קבלת הקישור
אחרי כמה דקות, תראה הודעה בירוק:
```
Your site is live at https://USERNAME.github.io/ramat-gan-bins/
```

**זה הקישור שלך!** 🎉

## שימוש באפליקציה

### לעובדים בשטח:
1. פתח את הקישור של `app.html` בטלפון
2. שמור את הדף על מסך הבית:
   - **iPhone**: Safari → כפתור שיתוף → "Add to Home Screen"
   - **Android**: Chrome → תפריט (3 נקודות) → "Add to Home screen"
3. לחץ על כפתור 📍 GPS למילוי אוטומטי של הכתובת
4. בחר רחוב, מספר בית, וספור פחים
5. לחץ "שמור דיווח"

### למנהל הפרויקט:
1. פתח את הקישור של `dashboard.html`
2. צפה בכל הדיווחים בזמן אמת
3. סנן לפי תאריך, רחוב, או מספר בית
4. ייצא לאקסל בלחיצה אחת
5. מחק דיווחים שגויים במידת הצורך

## פתרון בעיות

### GPS לא עובד בטלפון?
- ודא שנתת הרשאה לדפדפן לגשת למיקום
- ודא שאתה משתמש ב-HTTPS (הקישור מתחיל ב-https://)
- נסה בחוץ (לא בתוך בניין)

### לא רואה דיווחים בדשבורד?
- רענן את הדף
- בדוק שההרשאות ב-Firebase מאושרות:
  ```json
  {
    "rules": {
      "bin_counting": {
        ".read": true,
        ".write": true
      }
    }
  }
  ```

### רוצה לעדכן את האפליקציה?
1. עשה שינויים בקבצים
2. חזור ל-GitHub Repository
3. לחץ על הקובץ שרוצה לעדכן
4. לחץ על סימן העיפרון (Edit)
5. עשה שינויים
6. לחץ "Commit changes"
7. המתן 1-2 דקות לעדכון
